Arduino powered voice controlled car

December 10, 2023

This was a personal project I did using Arduino and C++. The car used a bluetooth module to connect to a mobile phone and the user can give commands to the car like 'go forwrd', 'turn right' via the mobile phone.

Skills I gained:

  • Hands on experience with Arduino
  • Introduction to C++
  • Knowledge on combining hardware and software
  • Electronics troubleshooting

Overall, this project strengthened my understanding of hardware-software interaction, real-time system control, and hands-on electronics troubleshooting, which are invaluable skills for any embedded systems
